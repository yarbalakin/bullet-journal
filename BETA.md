# План выхода на бета-тест

> Составлен: 2026-04-05

## Шаг 1 — Авторизация (2-3 дня)
- Подключить Supabase Auth
- Вход через Google + magic link на email
- Каждый пользователь получает свой изолированный аккаунт

## Шаг 2 — Перенос хранилища (1-2 дня)
- Данные из n8n → Supabase
- Авто-синк при каждом изменении
- Офлайн-режим остаётся через IndexedDB

## Шаг 3 — Домен (1 день)
- Зарегистрировать домен (bujo.app или bujojournal.ru)
- Настроить GitHub Pages на домен
- SSL автоматически

## Шаг 4 — Онбординг (1-2 дня)
- Экран приветствия при первом входе
- 3-4 слайда "что здесь есть"
- Тестовые данные чтобы не пустой экран

## Шаг 5 — Подготовка бета-группы (параллельно)
- Катя зовёт 5-10 подруг/знакомых
- Создать чат для фидбэка (Telegram)
- Список вопросов: что непонятно, что нравится, чего не хватает

## Что НЕ нужно для бета-теста
- Оплата и подписка — бета бесплатная
- App Store — PWA, просто ссылка
- AI-обложки — следующий этап (MVP-2)
